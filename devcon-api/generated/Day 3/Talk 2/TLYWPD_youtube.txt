Ethereum consensus pyspec (https://github.com/ethereum/consensus-specs) is an executable specification that serves as a reference for consensus layer devs. It is also used for creating the test case vectors for client teams.

In this lightning talk, we will have a tutorial on how to run the Ethereum consensus pyspec. And bonus, learn how to write a test by yourself!

Speaker(s): Hsiao-Wei Wang
Track: Layer 1 Protocol
Keywords: pyspec,consensus,beacon chain
